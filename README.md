## 🌐 GitHub Pages
1. Push all files to `main` branch root
2. Settings → Pages → Source: main / root
3. Add custom domain + update `CNAME` file
4. Point DNS CNAME → `yourusername.github.io`

No build step. Pure HTML/CSS/JS.
