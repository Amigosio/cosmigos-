## 🌐 GitHub Pages Setup

### Option A: Root deployment
1. Push all files to the `main` branch root
2. Go to **Settings → Pages → Source: Deploy from branch → main / root**
3. Add your custom domain in the **Custom domain** field

### Option B: Custom domain (your own)
1. Create a `CNAME` file in root with just your domain: `yourdomain.com`
2. In your DNS provider, add:
   - Type: `CNAME` → `yourusername.github.io`
   - (or A records pointing to GitHub Pages IPs)
3. Enable **Enforce HTTPS** in Pages settings
